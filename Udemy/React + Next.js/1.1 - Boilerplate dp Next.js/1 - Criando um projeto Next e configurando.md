## Usando o .editorconfig
Ao criar o arquivo `.editorconfig` pode definir configurações lá
```.
root = true

  

[*]

indent_style = space

indent_size = 2

end_of_line = lf

charset = utf-8

trim_trailing_whitespace = true

insert_final_newline = true
```

Link para esse editor: [editor config](https://editorconfig.org)
